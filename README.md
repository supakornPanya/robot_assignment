# Robot Framework Assignment

## Project Overview
This project is a Robot Framework-based automation suite designed to test and validate specific functionalities. The project structure includes test cases, resources, and variables to ensure modularity and reusability.

## Project Structure
```
main.robot
README.md
Resource/
	function.resource
	variable.resource
```

### Files and Directories
- **main.robot**: The main test suite containing test cases.
- **log.html**: Execution log generated after running the tests.
- **output.xml**: XML output file generated by Robot Framework.
- **report.html**: Test execution report.
- **Resource/**: Contains reusable resource files:
  - `function.resource`: Contains keywords and reusable functions.
  - `variable.resource`: Contains variables used across test cases.

## How to Run the Tests
1. Open a terminal and navigate to the project directory.
2. Run the following command:
   ```
   robot .\main.robot
   ```
3. After execution, check the `log.html` and `report.html` files for detailed results.

## Prerequisites
- Install Python (3.7 or later).
- Install Robot Framework:
  ```
  pip install robotframework
  ```

## Additional Notes
- Ensure all dependencies are installed before running the tests.
- Modify the `variable.resource` file to update test data as needed.
